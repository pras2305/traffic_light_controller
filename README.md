# Traffic Light Controller using Verilog

## Description
Designed and implemented a Traffic Light Controller using Verilog HDL based on a Finite State Machine (FSM).

## Features
- Four-state FSM
- Controls two-road traffic
- Simulated using EDA Playground
- Verified using a Verilog testbench

## Tools Used
- Verilog HDL
- Icarus Verilog
- EPWave
- EDA Playground

## Outputs
Road A:
- Green
- Yellow
- Red

Road B:
- Green
- Yellow
